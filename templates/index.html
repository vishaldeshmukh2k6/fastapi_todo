API Endpoints

  • GET /todos/ — List all todos
  • GET /todos/<id> — Get todo by ID
  • POST /todos/ — Create a new todo (use Postman or similar tool)
  • PUT /todos/<id> — Update a todo by ID (use Postman or similar tool)
  • DELETE /todos/<id> — Delete a todo by ID (use Postman or similar tool)
